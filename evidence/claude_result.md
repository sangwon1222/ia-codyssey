# CANIDAY 운영 회의록 정리

---

## 1. 회의 요약

- CANIDAY 앱은 출시 완료 후 초기 사용자 반응을 수집 중이며, 긍정·개선 피드백이 함께 확인되었다.
- SNS는 카니 캐릭터 기반 콘텐츠를 운영 중이며, 해시태그 다국어 관리 및 게시물당 5개 내외 사용을 확정했다.
- 크라우드 펀딩은 8월 완료를 목표로 하며, 펀딩 페이지 구성 요소(앱 소개·세계관·굿즈·리워드)를 확정했다.
- 결제 기능 도입을 위한 사업자 등록 필요성이 논의되었으나, 사업자 유형은 미확정 상태다.
- 사용자 피드백을 반영한 온보딩 화면 개선안을 다음 회의 전까지 정리하기로 했으며, 담당자는 미확정이다.

---

## 2. 결정사항

- ✅ 크라우드 펀딩 **8월 완료** 목표 설정
- ✅ 펀딩 페이지 구성: **앱 소개 / 카니 캐릭터 세계관 / 굿즈 구성 / 후원 리워드** 포함
- ✅ SNS 해시태그: **한국어·영어·일본어·중국어 간체·번체** 5개 언어로 관리, 게시물당 **5개 내외** 사용
- ✅ SNS 실제 업로드 빈도는 **제작 리소스에 따라 조정**하기로 함
- ✅ **온보딩 화면 개선안**을 다음 회의 전까지 정리하기로 함

> ⚠️ 아래 항목은 결정사항에서 제외
> - "카니가 한 짓일까?" 시리즈 집중 → **의견 단계, 미확정**
> - SNS 캐러셀 3개·릴스 2개·스토리 매일 1개 → **검토안, 미확정**
> - 사업자 등록 유형(개인/법인) → **미확정**

## 4. 리스크

- 🔴 **온보딩 개선 담당자 미지정** — 다음 회의 전 완료 기한이 있으나 담당자가 없어 실행 공백 발생 가능
- 🔴 **크라우드 펀딩 8월 목표** — 담당자·세부 일정·제작 리소스 미확정 상태로 일정 준수 리스크 존재
- 🟡 **사업자 등록 유형 미결정** — 결제 기능 도입 일정에 직접 영향을 미치므로 조기 결정 필요
- 🟡 **SNS 운영 리소스 불확실** — 기본 운영안(캐러셀 3·릴스 2·스토리 매일 1)이 검토 단계에 머물러 있어 실제 운영 기준 부재
- 🟡 **수치 데이터 부재** — 초기 사용자 반응 관련 정량 데이터가 공유되지 않아 의사결정 근거 부족

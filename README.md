# AI 도구 학습 과제

## 과제명
ChatGPT에게 일을 제대로 시키는 법 배우기

## 선택 과업
CANIDAY 운영 회의록 요약 및 Action Items 추출 자동화

## 과업 선정 이유
회의록 요약은 결정사항, 실행 항목, 담당자, 기한, 리스크를 정확히 구분해야 하는 업무다. 특히 회의 메모에 없는 정보를 AI가 임의로 생성하면 업무 혼선이 발생할 수 있으므로 환각 검증에 적합하다.

## 최종 산출물
1. LLM 모델 비교·선정 보고서
2. 시스템 설계 문서
3. 실행 로그
4. 최종 시스템 프롬프트
5. 모델별 원본 결과

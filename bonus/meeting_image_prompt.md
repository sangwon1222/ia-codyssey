CANIDAY 운영 회의 결과를 요약한 세로형 인포그래픽을 만들어줘.

스타일:
- 깔끔한 스타트업 보고서 스타일
- 부드러운 파스텔톤
- 모바일 앱 서비스 발표 자료 느낌
- 귀여운 캐릭터 브랜드에 어울리는 친근한 분위기
- 텍스트는 최소화하고 아이콘과 구조 중심으로 구성

화면 구성:
1. 상단 제목: CANIDAY 운영 회의 요약
2. 섹션 1: 결정사항
   - 펀딩 페이지 구성 확정
   - 해시태그 5개 언어 운영
   - 카니 시리즈 중심 운영
   - 개인사업자 등록
3. 섹션 2: Action Items
   - 펀딩 페이지 제작
   - 온보딩 개선안 정리
   - SNS 업로드 빈도 확정
4. 섹션 3: 확인 필요
   - 담당자 미지정
   - 정량 데이터 미공유
   - 제작 리소스 검토 필요

주의:
- 실제 수치가 없는 항목은 임의 수치를 넣지 말 것
- 복잡한 문장보다 짧은 키워드 중심으로 표현
- 회의록 요약용 시각 자료처럼 보이게 구성
